# Nameless Productions Website

A personal website showcasing projects and tools built by Nameless Productions. Built with Next.js 16, React 19, and Tailwind CSS 4.

## Features

- 📁 **Projects showcase** - Collection of interactive web projects:
  - **Cursed Calculator** - A calculator that returns random numbers after dramatic loading sequences
  - **Discord Webhooker** - A simple tool to send messages to Discord webhooks with support for embeds
- 🎨 **Dark theme** - Consistent dark gray theme throughout
- ⚡ **Fast & Modern** - Built on the latest Next.js App Router with React 19

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org) with App Router
- **UI Library**: [React 19](https://react.dev)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com)
- **Language**: [TypeScript](https://www.typescriptlang.org)
- **Package Manager**: [pnpm](https://pnpm.io)

## Getting Started

### Prerequisites

- Node.js 20 or higher
- pnpm (or npm/yarn/bun)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/nameless-Productions/website.git
cd website
```

2. Install dependencies:
```bash
pnpm install
```

3. Run the development server:
```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
